# Контекст проекта: LiteEditorAI

**Внимание для ИИ-ассистента:** Этот файл создан для того, чтобы быстро ввести тебя в курс дела. 

## Что это за проект
Это свежая копия репозитория `LiteEditorAI` (автор DanielLetto2020), скачанная с GitHub. 
Старая Git-история была удалена (`rm -rf .git`), и проект инициализирован с нуля (`git init`). 

**LiteEditorAI** — это десктопное приложение на базе фреймворка **Electron** (v33+). Оно выступает в роли «легкого редактора-оболочки для работы с ИИ-агентами в терминале» (например, с Claude Code).

## Архитектура и ключевые технологии
1. **Ядро (Backend):**
   - **Electron** (`main.js`, `preload.js`).
   - Сборка и упаковка через `electron-builder` (`package.json`).
   
2. **Интерфейс (Frontend):**
   - Лежит в папке `renderer/`, собирается через `esbuild`.
   - **CodeMirror 6**: Основной текстовый редактор с подсветкой синтаксиса и миникартой.
   - **Xterm.js**: Эмулятор терминала в браузере (с WebGL).

3. **Интеграция с системой (Киллер-фича):**
   - Используется **`node-pty`** (pseudo-terminal) для прямой двусторонней связи между браузерным терминалом (Xterm) и реальной консолью ОС (zsh/bash). Это позволяет гонять ИИ-агентов локально без API-ключей.

4. **Удалённое управление:**
   - Поддерживает мобильный пульт (Android) через WebSockets (`ws`). Файл `remote.js` выступает сервером.

5. **Дополнительно:**
   - Встроены коннекторы к БД (`mysql2`, `pg`, `sql.js`) и утилиты `ssh2` / `basic-ftp`.
